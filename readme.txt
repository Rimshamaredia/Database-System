This file contains instructions on how to compile/run the project

compilation using command line/terminal:
mac:
javac -cp .:postgresql-42.2.8.jar jdbcpostgreSQLGUI.java
pc:
javac -cp .;postgresql-42.2.8.jar jdbcpostgreSQLGUI.java

run using command line/terminal:
mac:
java -cp  .:postgresql-42.2.8.jar jdbcpostgreSQLGUI
pc:
java -cp  .;postgresql-42.2.8.jar jdbcpostgreSQLGUI
